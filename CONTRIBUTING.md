# 贡献指南（Contributing）

感谢你为 **Data Property and Compliance Skill** 做贡献。

## ⛔ 发布前提醒
本仓库当前处于 **发布暂缓** 状态（见 [`PUBLISH-HOLD.md`](./PUBLISH-HOLD.md)）：原始素材含有保密义务的内部培训题库。**在取得授权或完成净室重写前，请勿公开发布或推送到公开远程仓库。**

## 红线（务必遵守）
1. **不要提交任何保密/非公开材料**：内部培训题库、未公开的《工作指引(试行版)》原文、真实登记案例、真实申报材料 PDF、含真实主体信息的内容。
2. **不要把推断当法律结论**：涉及"是否违法/是否构成测绘"等定性，使用"可能涉及/宜由主管部门认定"的表述。
3. **所有法条引用必须可公开核验**：注明法规名称、版本、生效日期与官方出处链接（见 [`CITATIONS-REVIEW.md`](./CITATIONS-REVIEW.md)）。

## 怎么贡献
- **法规更新 / 条号纠错**：在 PR 描述里写明"法规名称 + 版本 + 生效日 + 官方链接"，并同步更新 `CITATIONS-REVIEW.md`。
- **新增审查工具 / checklist**：放在 `SKILL.md` 对应领域下，保持"方法论抽象、不含保密细则"。
- **代码类工具**（PDF 提取 / OCR / 校验脚本）：放在 `tools/`，遵循该目录的 **MIT** 许可（见 `tools/README.md`）。

## 许可与署名
- **内容**（`SKILL.md`、文档）：[CC BY 4.0](./LICENSE)。
- **代码**（`tools/`）：MIT。
- 提交 PR 即视为同意以上述对应许可证授权你的贡献。

## 提交规范
- 一个 PR 聚焦一件事；附上修改理由与出处。
- 涉及法律内容的改动，请在 PR 勾选："本改动不含任何保密/非公开材料"。
