# tools/ — 代码类工具（MIT）

本目录用于存放配套的**代码工具**（例如 PDF 文本提取、OCR 复核、表格矛盾校验、条号核对脚本等）。

## 双许可说明
- 本仓库**内容**（`/SKILL.md` 及文档）采用 **CC BY 4.0**。
- 本目录 **`tools/` 下的代码**采用 **MIT**（见 [`./LICENSE`](./LICENSE)）。

如此区分的原因：CC BY 更适合知识/文档，MIT 更适合代码复用。

## 现状
> 目前尚无代码文件。这是为未来脚本预留的结构。添加脚本时：
> - 放在本目录；
> - 顶部保留 MIT 头注释或在 `tools/LICENSE` 覆盖；
> - 不要在脚本里硬编码任何真实案例 / 保密数据；
> - 处理样例放 `tools/examples/`，且必须是**脱敏/虚构**数据。
